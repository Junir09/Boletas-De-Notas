# Assets

Recursos de estilos e imágenes del proyecto.

## css/
- `App.css`: Estilos globales, login y layout con sidebar.
- `index.css`: Estilos base del proyecto.

## images/
- `logo.svg`: Ícono/recursos gráficos.

## Uso
- Importa estilos en componentes: `import './assets/css/App.css';` o `import './assets/css/index.css';`.
- Importa imágenes: `import logo from './assets/images/logo.svg';`.
